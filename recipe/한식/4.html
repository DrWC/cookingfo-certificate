조리기능사 실기 레시피

한식

4. 국수장국


*재료

소면 80g, 쇠고기 50g, 달걀 1개, 애호박 60g(중 길이6cm), 석이버섯 5g(마른것, 잎이 넓은 것 1장), 실고추 1g, 식용유 5ml, 참기름 5ml, 소금 5g, 간장 10ml, 대파 1토막(흰부분 4cm 정도), 마늘 1쪽(중 깐 것)



*요구 사항


1. 고명[호박(돌려깎기)], 황.백 지단, 석이버섯의 크기는
0.2cm x 0.2cm x 5cm 로 만드시오.
2. 사리 1인분에 1.5배 분량의 장국을 붓고 오색 고명을 담아내시오.
3. 쇠고기는 육수를 내고 삶은 고기는 0.2cm x 0.2cm x 5cm 로 채썰어 고명으로 얹으시오.



*조리방법

1. 쇠고기는 핏물을 제거한 후 덩어리째 파, 마늘을 넣고 끓여 육수를 끓인다.
2. 석이버섯은 따뜻한 물에 불린다.
3. 달걀은 황/백으로 분리하여 소금을 넣고 풀어둔다.
4. 끓는 소금물에 국수를 넣고 삶아 찬물에 헹구어 사리를 튼다.
5. 애호박은 0.2cm x 0.2cm x 5cm 로 채 썬 후 소금에 살짝절였다가 물기를 제거한다.
6. 불린 석이는 손으로 비벼 씻어 이끼를 제거한 후 돌돌 말아채를 썬 후 소금, 참기름으로 간한다.
7. 지단은 식은 후 0.2cm x 0.2cm x 5cm 로 채 썬다.
8. 팬에 기름을 두르고 지단, 석이버섯, 애호박 순으로 볶아낸다.
9. 쇠고기편육을 0.2cm x 0.2cm x 5cm 로 채 썬다.
10. 실고추는 2cm 정도로 짧게 자른다.
11. 미리 끓여 놓은 육수를 간장, 소금을 넣어 색과 간을 맞춘 다음 끓인다.
12. 완성그릇에 국수사리를 담고 오색고명(편육, 호박, 황백지단, 석이, 실고추)을 얹은 뒤 1.5배 분량의 따뜻한 육수를 붓는다.



*유의 사항
1. 물이 끓으면 국수를 넣는데 이 때 부채처럼 펼쳐서 넣어야 서로 엉겨 붙지 않는다.
2. 끓어오를 때 찬물을 2~3번 넣어가며 끓이면 국수가 찰기 지고 빨리 삶아진다.
3. 컵에 찬물을 담아 국수를 한 두가닥 넣어 보았을 때 투명하게 보이면 익은 것이다.
4. 삶아진 국수는 찬물에서 비벼가며 여러 번 헹구어야 탄력이 있고 깔끔한 맛을 즐길 수 있다.
5. 사리를 튼 국수는 충분히 물을 빼 주어야지 육수를 부었을 때 흐트러지지 않는다.
