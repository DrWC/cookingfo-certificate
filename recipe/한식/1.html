조리기능사 실기 레시피

한식

1. 비빔밥


*재료

소고기(살코기, 30g), 쌀(150g), 애호박(6cm, 60g), 청포묵(6cm, 40g), 건다시마(5cm X 5cm, 1개), 고사리(불린것, 30g), 도라지(찢은것, 20g), 달걀(1개), 대파(흰부분10cm, 1토막), 마늘(중, 깐것 ,2쪽),검은후춧가루(1g), 고추장(40g), 깨소금(5g), 백설탕(15g), 소금(정제염 10g), 식용유(30ml), 진간장(15ml), 참기름(5ml)



*요구사항

가. 채소, 소고기, 황,백지단의 크기는 0.3cm x 0.3cm x 5cm로 써시오.(단, 지급된 재료의 크기에 따라 가감한다.)
나. 호박은 돌려깎기하여 0.3cm x 0.3cm x 5cm로 써시오.
다. 청포묵의 크기는 0.5cm x 0.5cm x 5cm로 써시오.
라. 밥을 담은 위에 준비된 재료들을 색 맞추어 돌려 담으시오.
마. 볶은 고추장은 완성된 밥 위에 얹어 내시오.



*조리방법

양념장 : 1. 쇠고기, 고사리양념(간장1T, 설탕1/2T, 참기름, 파, 마늘, 후추, 깨소금), 2. 약고추장(볶은고기, 고추장1T, 설탕1/2T, 참기름, 깨소금, 물1T)

1. 청포묵은 0.3cm x 0.3cm x 5cm로 썰어 끓는 물에 데치고, 찬물로 씻어 소금, 참기름으로 밑간한다.
2. 쌀과 동량의 물을 넣고 밥을 지어준다.
센 불로 끓이다가 밥이 끓기 시작하면 약불에서 7분정도 더 익혀 불을 끄고 뜸을 들인다.
3. 애호박은 돌려깍고 0.3cm x 0.3cm x 5cm로 채 썰고 소금을 뿌려 절인다.
4. 도라지는 0.5cm x 0.5cm x 5cm로 썰어 소름을 뿌려 절인다.
5. 고사리는 5cm채 썰고, 소고기 일부는 0.3cm x 0.3cm x 5cm로 썰어 양념한다.
6. 나머지 고기는 곱게 다져 약고추장에 사용한다.
7. 달걀은 황·백지단을 부쳐 0.3cm x 0.3cm x 5cm로 썬다.
8. 팬에 기름을 두르고 다시마를 튀긴다.
9. 다시마의 기름을 제거하고 손으로 부숴 준비한다.
10. 도라지, 애호박은 한번 행궈 물기를 제거하고, 도라지, 애호박, 고사리, 채 썬 고기 순으로 각각 볶는다.
11. 다진 고기를 볶다가 약고추장 양념을 넣고 볶는다.
12. 그릇에 밥을 편평히 담고 준비된 재료를 돌려 담아 마무리한다.



*유의사항

1. 밥은 질지 않게 .
2. 지급된 소고기는 고추장 볶음과 고명으로 나누어 사용한다.
